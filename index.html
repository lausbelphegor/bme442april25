EEG-Based Brain-Computer Interface Framework

Exploring advanced EEG decoding methods for BCI applications.

Introduction to EEG and BCI

BCI systems interpret brain signals to perform tasks. EEGs record electrical activity, crucial for BCIs.

Project Aim

  • Develop a robust EEG decoding model using convolutional and self-attention mechanisms.
  • Enhance BCI performance and user interaction.

Dataset Overview

Data from 50 subjects viewing images, recording EEG responses, aim to improve decoding accuracy.

System Input and Output

Input: Multi-channel EEG data. Output: Predicted cognitive states or object categories.

Neural Network Architecture

Description of the EEG Conformer architecture, combining convolutional and attention layers.

Expected Outcomes

Anticipate improvements in decoding speed and accuracy, facilitating more responsive BCIs.

Conclusion

Summarize the expected impact on BCI technology and future research directions.
